# sic-sim
A simple Circuit Simulator. Just describe netlist, and generate circuit diagram and circuit parameters

# Example NetList
```vi
-- This is example 1

V (12) (0, 1)
R (1) (1, 2)
R (2) (2, 3)
R (3) (3, 0)
R (4) (3, 0)
```sh
